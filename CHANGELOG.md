# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-01-20

### Added
- Initial release of claude-cortex
- Master index template (`MEMORY.md`) with hierarchical structure
- Activity log template for session tracking
- Project knowledge template with stack, architecture, and pattern sections
- Integration documentation template for external services
- Incident runbook template for bug fix documentation
- Feedback/preferences template for encoding coding style
- Reference/research template for archiving discoveries
- Two complete example memory setups (full-stack SaaS, mobile app)
- Setup script for one-command initialization
- Init script for per-project memory bootstrapping
- Sync script supporting Dropbox, iCloud, and Git backends
- Validation script for memory structure health checks
- Statistics script for knowledge base reporting
- Architecture documentation
- Best practices guide
- Common patterns catalog
- FAQ
- CLAUDE.md meta-instructions for self-referential memory usage
